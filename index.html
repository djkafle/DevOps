About DevOps

DevOps is a set of practices that combines software development (Dev) and IT operations (Ops). It aims to shorten the systems development life cycle and provide continuous delivery with high software quality.

Key DevOps Tools

  • Git: A distributed version control system for tracking changes in source code.
  • Jenkins: An open-source automation server which enables developers to build, test, and deploy their software.
  • Azure Container registry: A platform for developing, shipping, and running applications in containers.
  • Azure Webapp: An open-source system for automating the deployment, scaling, and management of containerized applications.
  • Ansible: An open-source tool for automation of software provisioning, configuration management, and application deployment.
